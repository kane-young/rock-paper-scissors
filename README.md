## Overview

<br>

**해당 프로젝트를 진행하면서 학습한 개념**

- commit 템플릿 생성, commit 메시지

- class, struct 의 차이
- class 상속 개념 (override 재정의)

-  CaseIterable, Comparable



- 참고사이트
